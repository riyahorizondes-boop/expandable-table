# Expandable Table - Angular

An enterprise data table built with Angular featuring:

- 📂 Expandable rows with child data loaded from JSON
- ✅ Multi-row selection with persistent highlight
- 🎨 Professional UI inspired by DataSphere Enterprise design
- 📊 Status badges and load metric bars
- 📄 Pagination

## Tech Stack
- Angular 17 (Standalone Components)
- TypeScript
- SCSS
- JSON data source

## Features
- Click the chevron (›) to expand/collapse child rows
- Click any row to select/deselect it
- Selected rows stay highlighted until clicked again
- Multiple rows can be selected simultaneously
